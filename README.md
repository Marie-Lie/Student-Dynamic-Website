# Student Dynamic Website

This project is a dynamic website developed as part of a web programming course.  
The website allows users to explore information about students and interact with the system through web forms and dynamic pages.

The goal of the project was to demonstrate how front-end and back-end web technologies work together to create an interactive website.

# Features
- Display student information through structured web pages
- User interaction through web forms
- Client-side functionality using JavaScript
- Server-side processing using PHP
- Styled interface using CSS

# Technologies Used
- HTML
- CSS
- JavaScript
- PHP
